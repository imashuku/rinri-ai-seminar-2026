## 各パート詳細

### 第1部｜現実を見る（10分）

1. **自己紹介 + アイスブレイク**
   - マツダ→博報堂29年→滋賀移住→独立
   - 「AIを使ったことがある方？」挙手
   - 今日のゴール：「帰るときに"自分こそAIを使うべきだ"と確信できること」

2. **中小企業の半数がAI未着手**
   - 東京商工リサーチ調査（2025年、6,645社）
   - 中小企業の50.9%が「AI方針すら未決定」
   - 大企業43.3% vs 中小企業23.4%の活用格差

3. **壁は「お金」でも「技術」でもない**
   - 最大の壁：「怖い」68.5%、「人がいない」55.1%
   - 「方針未決定」50.9%、「評価できない」43.8%、「用途不明」41.9%
   → **5つの壁が中小企業を止めている**

4. **お金より「目的」が成果を決める**
   - BuddieS調査（2025年、111社）
   - 10万円未満の投資でも効果実感率：100%
   - 10万円以上投資した企業は効果実感率80%に低下
   → **お金をかけた企業より、目的が明確な企業が勝つ**

### 第2部｜壁の正体（5分）

5. **5つの壁の根っこは1つ**
   - 怖い → 判断の軸がないから怖い
   - 人がいない → 学ぶ姿勢がないから人に頼る
   - 決められない → 自分の目的が見えないから決められない
   - 判断できない → 判断基準＝あり方がないから判断できない
   - 何に使うかわからない → そもそも何のために経営しているかが曖昧
   → **すべての根っこは「あり方」の不在**

6. **「あり方」は倫理。「やり方」はAI。**
   - わたしたちは「あり方」を日々磨いている
   - **すでにハンドルを握れる人間だ**

### 第3部｜あなたはすでに越えている（18分）
**壁1つにつき約3分。壁→データ→栞→越え方の順で語る**

7. **壁①怖い（68.5%）→ 第15条「信ずれば成り、憂えれば崩れる」**
   - 情報漏洩・失敗への恐怖が最大の心理的障壁
   - しかし恐れて止まっていること自体が最大のリスク
   - 「憂えれば崩れる」＝ 恐れて動かなければ、取り残される
   - 「信ずれば成り」＝ 10万円未満でも始めた企業は100%効果を実感
   - 皆さんは「信成万事」を知っている。だから恐れずに踏み出せる

8. **壁②人がいない（55.1%）→ 第4条「人は鏡、万象はわが師」**
   - 「専門人材がいない」が最大の導入障壁
   - しかし、AIそのものが「万象」の一つ。AIを師にすればいい
   - 人は鏡 → AIも鏡。問いかけがそのまま映る
   - 学ぶ姿勢がある人に、AIは最大の力を返す
   - 皆さんは「万象はわが師」と毎朝唱えている。その姿勢こそが壁を越える

9. **壁③決められない（50.9%）→ 第1条「今日は最良の一日、今は無二の好機」**
   - 半数の企業が「方針すら未決定」のまま止まっている
   - 「来月」「来期」「落ち着いたら」→ そうやって3年が過ぎた
   - 「今日は最良の一日」＝ 始めるのに一番いい日は今日
   - 皆さんは毎朝この言葉を実践している。だから今日決められる

10. **壁④判断できない（43.8%）→ 第3条「運命は自らまねき、境遇は自ら造る」**
    - 「良いか悪いかわからない」から動けない
    - しかし、自分の事業の判断は自分にしかできない
    - 「運命は自らまねく」＝ AIを使うかどうかも自分の選択
    - やってみて初めて判断できる。やらずに判断はできない
    - 皆さんは「境遇は自ら造る」を知っている。だから自分で判断できる

11. **壁⑤何に使うかわからない（41.9%）→ 第16条「己を尊び人に及ぼす」**
    - 「利用用途がない」＝ AIの目的がわからない
    - しかし目的は明快。**己を尊び、人に及ぼす**ために使う
    - 自分の時間と心の余裕を取り戻す（己を尊ぶ）
    - その余裕で家族・社員・地域に貢献する（人に及ぼす）
    - 皆さんはすでにこの目的を持っている。だから用途は自ずと見える

12. **決めゼリフ：5つの壁を越える力が、すでにわたしたちの中にある**

### 第4部｜実践事例（8分）

13. **一人会社の社長には、止めてくれる人がいない**
    - 15案件同時並行。締切漏れ、採算度外視、先送り
    - 一人だから気づいてくれる人がいない

14. **AIに「役職」を与えた ― AI取締役会**
    - CEO: 毎朝10秒で全案件把握
    - CFO: 採算チェック・投資判断（時給換算）
    - PM: 締切管理・「やらないこと」を決定
    - 取締役会: 4視点で新規案件の受け入れ判断
    - 一人会社なのに「取締役会」がある

15. **毎朝5分で、会社が回る**
    - ボイスジャーナル → AI振り分け → PM起動 → 今日やること3つ
    - 朝の案件把握: 30分 → 5分

16. **一番変わったこと：孤独な判断がなくなった**
    - 断れるようになった（CFOが数字で見せるから）
    - 重要だけど緊急じゃない仕事に時間が取れるようになった

17. **業務時間のBefore/After**
    - 提案書：3時間 → 30分
    - メール文案：20分 → 3分
    - 議事録整理：1時間 → 5分

### 第5部｜波紋とクロージング（9分）

18. **AIを使うことは、効率化ではない**
    - 生まれた時間を「大切なこと」に使う

19. **波紋の構造**
    - 自分 → 家族 → 職場 → 地域 → 日本
    - AIで余裕を取り戻す → 対話が増える → 会社が変わる → 地域が変わる

20. **倫理AI実践士**
    - 「私は実践する」と決めた瞬間から
    - 仲間と一緒に

21. **中野さんへバトンタッチ**
    - 「ここまでは"なぜ"と"何を"。ここからは"どうやって"」

# 草津市倫理法人会 自主企画セミナー報告書

## 「倫理×AI実践セミナー」企画全体像と第1回実施報告

草津市倫理法人会　副専任幹事　今宿裕昭
令和8年3月

---

## Executive Summary

草津市倫理法人会は、普及活動の一環として「倫理×AI実践セミナー」を企画した。AIという社会的関心の高いテーマを入口に、倫理法人会に接点のなかった経営者層にアプローチし、MS・研究所行事への参加につなげることが目的である。

本企画は**全2回完結**とし、第1回（3月実施済）で関心喚起、第2回（5月予定）でMS体験への橋渡しを行う構成とする。

第1回は54名が参加し満足度4.62/5.0を記録したが、入会・MS参加といった普及の具体的成果はこれからであり、第2回までにゲストへのフォローを進める。

---

## 1. 企画の目的と位置づけ

### 1.1 解決すべき課題

草津市倫理法人会の普及活動において、以下の課題が存在する。

- **倫理法人会を知らない経営者層**へのアプローチ手段が限られている
- MSへの初回参加のハードルが高い（早朝・内容が未知）
- 既存の普及活動だけでは接点を持てない層がいる

### 1.2 企画の位置づけ

| 項目 | 内容 |
|------|------|
| **位置づけ** | MS・研究所行事を補完する普及活動 |
| **目的** | 未会員経営者の関心喚起 → MS参加 → 入会 |
| **本企画の役割** | 普及導線の「入口」。本筋はあくまでMS・研究所行事 |

> **本企画は、MSや研究所行事に代わるものではなく、そこへの参加者を増やすための入口として設計した。**

## 3. 第1回 実施報告

### 3.1 開催概要

| 項目 | 内容 |
|------|------|
| **日時** | 令和8年3月17日（火）18:30〜20:30 |
| **会場** | キラリエ草津 402号室 |
| **講師** | 今宿裕昭 / 中野充博（株式会社バーグ企画） |
| **参加者** | 54名（会員47名 + ゲスト7名） |
| **参加費** | 無料 |
| **経費** | 7,500円 |

### 3.2 内容構成

中小企業のAI導入を阻む「5つの壁」を、万人幸福の栞の教えと対応させて解説する構成とした。

| 壁 | 対応する栞の教え |
|---|---|
| 怖い（情報漏洩・失敗への不安） | 第15条「信ずれば成り、憂えれば崩れる」 |
| 人がいない（専門人材不在） | 第4条「人は鏡、万象はわが師」 |
| 決められない（方針未決定） | 第1条「今日は最良の一日」 |
| 判断できない（評価不能） | 第3条「運命は自らまねき、境遇は自ら造る」 |
| 何に使うかわからない | 第16条「己を尊び人に及ぼす」 |

> 単なるAI技術論ではなく、「倫理を学んでいる人こそAIを使いこなせる」というメッセージを軸とした。

### 3.3 アンケート結果（回答者29名）

#### 満足度

| 指標 | 結果 |
|------|------|
| **総合満足度** | **4.62 / 5.0** |
| 満足度4以上の割合 | 93.1%（27名） |
| AI活用意欲 | 4.41 / 5.0 |
| 次回セミナー参加希望 | 79.3%（23名） |

#### 最も印象に残った内容

| 内容 | 回答数 | 割合 |
|------|--------|------|
| 実践事例（AI取締役会） | 21名 | 72.4% |
| 倫理の教えとAIのつながり | 4名 | 13.8% |
| 5つの壁フレームワーク | 3名 | 10.3% |

#### ゲスト参加者の反応（ゲスト7名中、アンケート回答2名）

| 指標 | ゲスト回答者（n=2） | 全体（n=29） |
|------|-----------------|-------------|
| 満足度平均 | 4.00 | 4.62 |
| AI活用意欲平均 | 3.50 | 4.41 |
| 次回セミナー希望 | 100%（2/2） | 79.3% |
| 入会への関心 | 0名 | 1名 |

- ゲスト7名が参加。うちアンケート回答は2名（回答率29%）
- 回答者2名とも次回セミナーへの参加を希望
- 1名（東氏）は「AIに否定的だったが、活用すべきと感じた」と意識変容が発生
- 入会への関心を示したゲストはまだいない。未回答の5名を含め、第2回までのフォローが必要

#### 参加者からの改善要望

- 具体的なツール名・操作方法の説明がほしい
- 「倫理の実践としてAIを使う」の説明をもう少し詳しく
- AI未経験者向けのフォローが必要

### 3.4 第1回の評価

| 項目 | 達成状況 |
|------|----------|
| ゲスト参加者の集客 | **○** ゲスト7名が参加（参加者54名中） |
| 関心の喚起 | **○** 満足度4.62、次回希望79.3% |
| 倫理×AIの接点提示 | **△** 一部参加者から説明不足の声あり |
| **普及成果（入会・MS参加）** | **これから** |

## 4. 第2回に向けた取り組み

### 4.1 第1回→第2回の間に行うこと

1. **ゲスト参加者へのMSお声がけ** — 個別にフォローし、MS体験参加を促す
2. **第1回の改善点反映** — ツール紹介・初心者向け内容の充実
3. **第2回の構成にMS体験導線を組み込む** — 「次はMSで会いましょう」で終わる設計

### 4.2 第2回の成功指標

| 指標 | 目標 |
|------|------|
| ゲスト参加者のMS体験参加 | 具体的な申込を得る |
| 入会見込み | 1名以上 |

---

## 5. まとめ

- 本企画は**全2回完結の普及活動**であり、MS・研究所行事を補完する位置づけ
- 第1回で関心喚起に成功（54名参加、満足度4.62）
- 普及の具体的成果（入会・MS参加）はこれからであり、ゲストフォローを進める
- 第2回でMS体験への橋渡しを行い、企画を完結する
